# JS Code Challenge

### Install

```sh
npm install
# or
yarn
```

### Run the app

```sh
npm start
# or
yarn start
```

Then application is available on `http://localhost:8080`.

### Run the tests

```sh
npm test
# or
yarn test
```

#### UI Component: ProductDisplay
```sh
  src/app/components/ProductDisplay/ProductDisplay.test.js
```

#### Helpers tests

```sh
  src/app/helpers/helpers.test.js
```
